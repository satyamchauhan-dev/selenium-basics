# Selenium Basics
This repository contains basic Selenium scripts created for learning browser automation.

## What this project covers
- Launching a web browser using Selenium
- Navigating to a website
- Performing simple user interactions such as search actions
- Closing the browser after execution

## Technologies Used
- Python
- Selenium WebDriver
- ChromeDriver

## Purpose
This project focuses on understanding the fundamentals of automation testing and browser interaction using Selenium.
